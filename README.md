# go-calc
Modern Desktop Calculator written in Golang, HTMl/CSS, Javascript, and utilizes Webview

## Install
```
$ git clone https://github.com/luxunator/go-calc
$ cd go-calc
$ go get github.com/webview/webview
```

## Build
```
$ go build -o /path/outputfile
```

## Run
```
$ cd /path
$ ./outputfile
```
